# Privacy Policy for Robo-Stream Client

Last updated: January 26, 2026

## Data Collection

Robo-Stream Client does not collect, store, or transmit any personal information or user data to external servers.

## Local Storage

The app stores the following data locally on your device:
- Server URL you configure
- Session information for server connection
- App preferences

This data never leaves your device and is not shared with any third parties.

## Network Communication

The app communicates only with the server URL you provide, which should be a server on your local network running the RoboStream Server software.

No data is transmitted to our servers, analytics services, or third parties.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted at this URL.
