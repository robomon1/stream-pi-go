APP ICONS NEEDED
================

Please add your app icons to this directory:

Required files:
- icon-192.png (192x192 pixels)
- icon-512.png (512x512 pixels)

These icons are used for:
1. PWA installation (Add to Home Screen)
2. App icon on mobile devices
3. Splash screens

You can generate these from a single image using:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Recommended design:
- Simple, recognizable icon
- Works well at small sizes
- Solid background color
- High contrast

Example design ideas:
- Play button (▶)
- Recording dot (●)
- OBS logo variant
- Remote control icon
- Grid of buttons
